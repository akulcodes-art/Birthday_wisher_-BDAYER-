Steps to Build the Project
1. Design the UI

Designed the interface and user flow using Figma.

Created layouts for:

Homepage

Birthday wish form

Dashboard showing birthday statistics

Planned the user experience before starting development.

2. Setup the React Project

Created the project using React in Visual Studio Code.
3. Setup Backend Services

Integrated Firebase to handle:

User data storage

Birthday records

Real-time database features

Steps:

Create a Firebase project.

Enable Firestore / database.

Connect Firebase to the React app using the Firebase SDK.

4. Implement Email Notifications

Used EmailJS to send birthday wishes automatically.

Process:

Create an EmailJS account.

Configure an email template.

Connect EmailJS API to the React application.

Trigger emails when a birthday is detected.
